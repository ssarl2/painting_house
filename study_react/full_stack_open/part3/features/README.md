## Install
`npm install --no-audit`  

## Run
`npm start`  

## Dev
`npm run dev`  

## Test
`npm test`  

## Build frontend
`npm run build:ui`  

## Deploy backend and updated frontend for Render
`npm run deploy:uitest`
