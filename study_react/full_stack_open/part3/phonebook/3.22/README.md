## Install
`npm install`  
`npm run build:ui`

## Run
`npm start`  

## Dev
`npm run dev`  

## Lint
`npm run lint`  
`npm run lint:fix`  

## Deploy
`npm run deploy`  
And in render, change the root directory path in Settings
